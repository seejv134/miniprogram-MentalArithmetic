# miniprogram-MentalArithmetic

WeChat mini program **Daily Mental Arithmetic** (日常口算) for everyday practice.

## WeUI (`weui/`)

This project may include [Tencent WeUI](https://github.com/Tencent/weui) under `weui/` as a **development reference only** (styles, examples, documentation). It is **not** a runtime dependency of the mini program—you do **not** need to obtain it to build or run this project.

If you want a local copy for reference:

- **With Git submodules** (after cloning this repo):  
  `git submodule update --init --recursive`
- **Clone from scratch** (includes submodules):  
  `git clone --recurse-submodules <this-repo-url>`
- **Manual**: clone `https://github.com/Tencent/weui.git` into the `weui` directory at the repository root.
