# miniprogram-MentalArithmetic

微信小程序「日常口算」，用于日常口算练习。

## WeUI（`weui/`）

本仓库可能在根目录下的 `weui/` 中包含 [Tencent WeUI](https://github.com/Tencent/weui)，**仅作开发参考**（样式、示例、文档等），**不是**小程序运行时的必需依赖；**不获取该目录也可以正常开发与运行**。

若你需要本地参考，可通过以下方式获取：

- **使用 Git 子模块**（克隆本仓库之后）：执行  
  `git submodule update --init --recursive`
- **首次克隆时一并拉取子模块**：  
  `git clone --recurse-submodules <本仓库地址>`
- **手动**：将 `https://github.com/Tencent/weui.git` 克隆到仓库根目录下的 `weui` 文件夹。
